Reads a Netplex configuration file:

           <settings>

The configuration options are now documented in [root:Netplex_admin].

More documentation: See [root:Netplex_intro].webserver for a complete example of a config file. See [root:Netplex_intro].crsock for explanations how to specify sockets in the config file.
